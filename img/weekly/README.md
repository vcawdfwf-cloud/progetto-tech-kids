# Carpeta `weekly`

Coloca aquí una imagen para la "Imagen de la semana".

Formato recomendado del nombre de archivo: `YYYY-MM-DD.jpg` o `YYYY-MM-DD.png` donde `YYYY-MM-DD` es la fecha del lunes de la semana (por ejemplo `2026-01-05.jpg`).

Prioridad de uso por `js/main.js`:
1. `img/weekly/YYYY-MM-DD.jpg`
2. `img/weekly/YYYY-MM-DD.png`
3. `img/weekly.jpg` o `img/weekly.png` (opcional, fallback global)
4. `img/weekly.url` (archivo de texto con una URL externa)
5. Unsplash Source (fallback automático)

Agrega tus imágenes en esta carpeta y la página las mostrará automáticamente la semana correspondiente.
